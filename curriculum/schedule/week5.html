Chain Rule

How do I differentiate the composition of two functions?

Suppose \(f\) is differentiable at \(a\) and \(g\) is differentiable at \(f(a\). Then \(\left(g \circ f\right)\) is differentiable at \(a\), and

\(\left(g \circ f\right)'(a) = g'(f(a))\, f'(a)\).
This is the celebrated  chain rule and you can read more about it on  Page 79 of the Textbook.

You might wonder why you'd care about such a rule: a couple examples where it helps include  differentiating \((1+2x)^5\) and \(\sqrt{x^2+0.0001}\). The slogan “derivative of the outside function at the inside, times the derivative of the inside” will, I hope, make it easy to remember what to do! It takes practice.

How do I find the tangent line to a curve?

Sometimes, we're given equations like \(y^2 - x^3 + x - 1 = 0\); we can plot the points satisfying this equation in the plane, but the resulting curve fails the vertical line test—so there's no function which has that whole curve as its graph. Nevertheless, we can still ask for a tangent line to the curve: the trick is  implicit differentiation, which you can read about on  Page 86 of the Textbook.

There's a bunch of fascinating curves in the plane, but one example stands out historically:  the folium of Descartes, given by \(x^3 + y^3 - 3axy = 0\) for some choice of \(a\). Descartes challenged Fermat to find the tangent line to the curve; Fermat solved the problem, and now so can you, using implicit differentiation.

How do I find the derivative of an inverse function?

Given an invertible function \(f\), the derivative measures how changing the input affects the output. The derivative of the inverse \(f^{-1}\) encodes how wiggling the output of \(f\) would “affect” the input to \(f\), so perhaps it is reasonable to expect that

\( \left( f^{-1} \right)'(y) = \displaystyle\frac{1}{f'(f^{-1}(y))} \).
This is made precise with the  inverse function theorem.

As a particular example, the inverse of the function \(f(x) = e^x\) is the natural logarithm. We  compute the derivative of log and find that

\(\displaystyle\frac{d}{dx} \log x = \displaystyle\frac{1}{x}\).
Note that I follow the convention that there is only one log—the natural log—so I write it as \(\log x\); sometimes you'll see this function denoted \(\ln x\) instead.

Once we know that \(\displaystyle\frac{d}{dx} \log x = \displaystyle\frac{1}{x}\), there is one fun application of this fact:  logarithmic differentiation, which you can read more about on  Page 91 of the Textbook.

How can I multiply quickly?

Multiplication is an extremely important operation, so we should  figure out how to multiply quickly. Thousands of years ago, the Babylonians used  quartersquares which makes multiplication faster, but requires a separate table for division. Logarithms are even better since logs, for instance, not only turn multiplication into addition, but division into subtraction. You can play around with logarithms by printing out your very own  log table or  slide rule.

How do I justify the derivative rules?

We've been using the power rule, but so far, we only checked it for positive, whole number exponents;  how do we justify the power rule in general? One way is with logarithms! Because logarithms convert multiplication into addition, we can also use logs to give  another proof of the product rule.

Last week, in Week 4, we introduced the quotient rule but we didn't prove it; with the product rule and chain rule and power rule, we can now give  a proof of the quotient rule.
