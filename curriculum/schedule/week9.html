What is linear approximation?

This week is called linear approximation, so what does that even mean? The basic idea is the formula

\( f(x+h) = f(x) + h f'(x) \)
which  comes from the definition of the derivative. By using linear approximation, you can approximate the change in a function without actually evaluating the function: a classic real-world example is approximating the  volume of an orange rind.

You can read more about this on  Page 153 of the Textbook. We also did some exercises along these lines during  Week 3 on mooculus.

What happens if I repeat linear approximation?

The derivative tells you how wiggling the input affects the output; if you do enough wiggling, you can wiggle your way rather far to the right! In other words, by performing  repeated linear approximation, otherwise known as the Euler method you can approximate a function, given information about its derivative. To see this sort of technique in action,  compute the natural logarithm of two and three, and then read on  Page 161 of the Textbook.

These ideas also appear in the during  slope field series of exercises on mooculus.

What does \(dx\) mean by itself?

Another way to talk about linear approximation is via  differentials, like \(dx\). You can find more about this on  Page 155 of the Textbook.

What is Newton's method?

The derivative can help us approximate zeroes of functions, as well, via  Newton's method.. If you're standing near a zero of a function, you can replace the function with the tangent line approximation, figure out where the tangent line crosses the \(x\)-axis, and then use that point as a hopefully better approximation to where the function crosses the \(x\)-axis. This technique is described—with pictures!—on  Page 158 of the Textbook.

There's a couple nice applications of Newton's method. The obvious one is to approximate roots of polynomials! If I were trying to solve $$ax^2 + bx + c = 0$$, I'd just use the quadratic equation, but if I try to solve $$x^5 + x^2 -1 = 0$$, I'm in trouble, because by Galois theory, that polynomial's root cannot be expressed using whole numbers, the usual arithmetic operations (plus, subtract, multiply, divide) and exponents. Nevertheless, with Newton's method we can  approximate a point \(x\) so that \( x^5 + x^2 - 1 = 0 \).

You can try approximating a root of a polynomial yourself by doing the  Newton's method exercise on mooculus.

Newton's method is the basis of  Newton-Raphson division, which is a method for dividing without dividing: by which I mean, you can approximate $$1/b$$ by adding, subtracting and multiplying repeatedly.

What is the mean value theorem?

The  mean value theorem is the main technical tool for connecting information about the derivative to the original function. In words, it says that, given a secant line to a graph, there is a tangent line with that same slope. Or more precisely,

If \(f\) is continuous on the interval \([a,b]\) and differentiable on \((a,b)\), then there is a point \(c\) so that \(f'(c) = \displaystyle\frac{f(b) - f(a)}{b-a}\).
You can find a proof on  Page 165 of the Textbook. This ends up having some nice applications, like a  proof that $$f'(x) > 0$$ implies $$f$$ is increasing.

Many textbook problems on the mean value theorem emphasizing finding the point $$c$$ where $$f'(c)$$ equals $$\frac{f(b) - f(a)}{b-a}$$, but  it is better to regard the mean value theorem as an existential result.
