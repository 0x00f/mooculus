This week we begin antidifferentiation.  What is antidifferentiation? It is the process of untaking derivatives, of finding a function whose derivatives is a given function. Since it involves working backwards, it sometimes feels like unbreaking a vase.

But there are so many antiderivatives!

The first of many problems with antidifferentiation is that  there is more than one antiderivative. If $$F'(x) = f(x)$$, then $$F$$ is an antiderivative for $$f$$, but so would be the function $$G$$ given by $$G(x) = F(x) + 17$$, since $$G'(x) = f(x)$$. To deal with this, we ask for the most general antiderivative and write, for instance, $$F(x) + C$$ for an unspecified constant $$C$$.

You can get a first taste of antidifferentiation by  drawing some antiderivatives on mooculus and you can also start reading  Page 170 of the Textbook which begins the chapter on antidifferentiation.

How am I supposed to compute antiderivatives?

Often computing antidifferentiation requires some real insight, but there are some rules that help. The first is that  the antiderivative of a sum is the sum of antiderivatives. In other words,

$$\displaystyle\int \left( f(x) + g(x) \right) \, dx = \displaystyle\int f(x) \, dx + \displaystyle\int g(x) \, dx$$.
We haven't previously mentioned this notation: we write $$\displaystyle\int f(x) \, dx$$ for the antiderivative of $$f$$. Sometimes antiderivatives are called “indefinite integrals” and the process of antidifferentiation is called “indefinite integration.” The long s-shaped $$\displaystyle\int$$ is an integral sign, and the reason that it does double duty as an antidifferentiation sign boils down to the biggest theorem in the course, the the fundamental theorem of calculus, which we won't get to for a couple weeks.

There are some other rules, too. For instance, to  antidifferentiate $$x^n$$ you can use

$$\displaystyle\int x^n \, dx = \frac{x^{n+1}}{n+1} + C$$ provided $$n \neq -1$$.
This is the power rule running in reverse. In the case $$n = -1$$, one must  be a bit more careful about handling constants, though it is enough to remember
$$\displaystyle\int \displaystyle\frac{1}{x} \, dx = \log |x| + C$$.
We can  antidifferentiate trigonometric functions and  antidifferentiate $$e^x$$ and $$\log x$$, too.

You can get some practice by  antidifferentiating polynomials on mooculus and  practicing running derivatives in reverse.

This is so hard!

Antidifferentiation really is harder than differentiating! One analogy to make is to multiplying and antimultiplying, which normal people would call factoring.  It is “easy” to multiply, and hard to factor, which is somewhat analogous to the difficulties of antidifferentiation.

In fact,  sometimes we cannot write down the antiderivative, even when it exists. Think back to Galois theory from last week, where we learned that a certain polynomial's root cannot be expressed using whole numbers, the usual arithmetic operations (plus, subtract, multiply, divide) and exponents—and yet, that polynomial does have a root! We just can't write it down.

But not all hope is lost. There are some general tricks, like  antidifferentiating $$f(mx+b)$$ which will develop into fancier and fancier methods as we progress in the course. This trick for $$f(mx+b)$$ can be  practiced on mooculus.

Why would anyone want to do this?

Sometimes we only have information about the derivative, and we want to recover information about the original function. Antidifferentiation can help solve those problems! The prototypical examples are  starting with velocity and recovering position or  starting with acceleration data and recovering position.

And I do think some of the problems are simply fun: a fun example to think about  antidifferentiating sine squared. You can see these problems visually by  considering slope fields.
