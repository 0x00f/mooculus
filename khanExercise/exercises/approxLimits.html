Consider a function f(x). While you don't know the formula for f(x) you can nevertheless still evaluate it for given values of x, provided that x \ne LIMITX. Find the value of

\displaystyle\lim_{x\to LIMITX}f(x)

by evaluating f(x) at various values, or state that the limit does not exist (DNE).

answer

We should check values near, but not equal to LIMITX.

Try x = LIMITX-.1.

Try x = LIMITX+.1.

Remember for the limit to exist, the limit from the left must equal the limit from the right.

The limit is answer.
